# 羽毛球接龙统计

一个用于群内管理羽毛球活动的小型网页应用，支持注册、发起接龙、参与统计等功能。

## 功能
- **用户注册/登录**：使用手机号与密码管理账号。
- **发起接龙**：填写打球日期后创建接龙，其他成员可加入。
- **参与统计**：基于接龙参与记录，统计每位用户的参与次数。
- **首页展示**：展示当年所有接龙及年度参与统计概览。
- **统计视图**：独立统计页，可按年或按月查看参与次数排行榜。

## 本地运行
1. 创建虚拟环境并安装依赖：
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
2. 启动应用：
   ```bash
   flask --app app run
   ```
3. 浏览器访问 `http://127.0.0.1:5000`。

> 首次运行会自动创建本地 `app.db` SQLite 数据库。
